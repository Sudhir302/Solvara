## 🚀 Getting Started
### Prerequisites
- node
- vs code
- git

### Installation
1. Clone the solvara repo:
    ```
    git clone https://github.com/Sudhir302/Solvara.git
    ```
2. Navigate to the client directory:
    ```
    cd client
    ```
3. Install dependencies:
    ```
    npm install
    ```
4. Run the client
    ```
    npm run dev
    ```
